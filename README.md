DUU - Daily-Use-Utilities.
To easily launch DUU use "DUU-{Platform}", or "DUU-Windows.bat" or "DUU-Linux.sh", or original "DUU-Launcher_Windows.bat" or "DUU-Launcher_Linux.sh".
Yes, just like that. :3

There's these utilities:
- CleanTemp to clean the Temp folder.
- docxANDpdf to convert .docx file to .pdf file or .pdf file to .docx file.
- folderTOzip to convert folder to .zip file
- pngANDjpg to convert .png file to .jpg file or .jpg file to .png file.
- SystemCheck to check and repair System files.

Dependencies:
- Python, to run some scripts.

First two numbers in version means some actual change.
Third number means a little change.
